# socketio-monitor
> Simple socket.io server to monitor streaming data

Your data provider -> POST /socketio-monitor/data -> browser

## Getting started

 - Install npm (node).
 - To install packages. Go to root directory of the project and run
```shell
npm install
```
 - To start server
```shell
node index.js
```
